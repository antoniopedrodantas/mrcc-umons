This web app was developed by António Dantas and Bernardo Moreira for the Multimedia Retrieval and Cloud Computing course at the University of Mons. The goal of the project is to develop a method that extracts features from an image database to later use them to compare their similarity. After the cloud deployment this results in a platform the serves as an image search engine using artificial intelligence.
